# Security

## Reporting a vulnerability

Please **do not** open a public issue for security reports.

- Open a **private security advisory** on GitHub (if enabled for this repository), or
- Email the maintainer (add your contact when you fork / own the repo).

Include steps to reproduce, affected versions, and impact if known.

## Scope

This project shells out to the official Graphite CLI (`gt`). Vulnerabilities in `gt` or Graphite’s services should follow Graphite’s guidance: [Privacy & security](https://graphite.dev/docs/privacy-and-security) and [Graphite](https://graphite.dev/).
